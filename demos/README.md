# RLGym2D 演示视频

这个目录包含了RLGym2D各个任务的演示视频。

## 📹 视频列表

### 1. cartpole_demo.mp4
- **任务**: 倒立摆平衡控制
- **描述**: 智能体控制小车左右移动，保持摆杆直立
- **训练时间**: ~5分钟
- **成功标准**: 保持1000步不倒下

### 2. walker_demo.mp4  
- **任务**: 火柴人行走
- **描述**: 智能体控制火柴人协调四肢进行双足行走
- **训练时间**: ~30分钟
- **成功标准**: 向右行走超过5米

### 3. arm_demo.mp4 (待录制)
- **任务**: 机械臂目标到达
- **描述**: 智能体控制2关节机械臂到达目标点
- **训练时间**: ~10分钟
- **成功标准**: 末端距离目标小于10像素

## 🎬 如何录制新视频

```bash
# 1. 训练模型
python train.py --task cartpole --envs 8

# 2. 录制演示视频
python play.py --task cartpole --model runs/cartpole_xxx/best_model/best_model.zip --record --episodes 3

# 3. 视频保存在 videos/ 目录中
# 4. 复制最佳视频到 demos/ 目录
cp videos/cartpole_xxx/cartpole_demo-episode-0.mp4 demos/cartpole_demo.mp4
```
